# MintMarket

### An application built using React native which can transfer and store NFTs, mint them and connect wallets.


#### Technologies Used: 
  <ul>
    <li>React Native --> Framework used to make the application. </li>
    <li>Firebase --> used for user authentication</li>
    <li>WalletConnect --> used to connect wallets and manage multiple wallet connections</li>
    <li>Solidity --> to write smart contracts to handle NFT minting and transactions.</li>
    <li>Remix IDE --> to test and develop smart contracts</li>
  </ul>

  #### Running the Project locally 
  The project was made using expo, to run the project locally you can follow the official  <a href="https://reactnative.dev/docs/environment-setup" >documentation</a> to set up the local development environment.
  After setting up the environment for development follow the following steps 

  ```
  git clone repo
  cd repo
  npx expo start 
```
